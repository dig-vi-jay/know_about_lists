ORDERED LIST

HTML Ordered List or Numbered List displays elements in numbered format. The HTML ol tag is used for ordered list. We can use ordered list to represent items either in numerical order format or alphabetical order format, or any format where an order is emphasized.An ordered list created using the ol element, and each list item starts with the li element. Ordered lists are used when the order of the list's items is important. The list items in an ordered list are marked with numbers.
